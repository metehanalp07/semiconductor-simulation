## 📋 Overview

This simulation visualizes the behavior of free carriers (electrons and holes) in silicon (Si) in real time. Thermal generation, drift current, diffusion current, and recombination processes are animated based on physical formulas.

Opens directly in any browser — no installation required.

---

## 🚀 How to Use

Open the file in any modern browser:
```
semiconductor_simfinalized.html
```

## ⚛️ Physical Background

### Core Concepts

#### Electrons and Holes
- **Electron:** A negatively charged carrier that jumps from the valence band to the conduction band when thermal energy breaks a covalent bond.
- **Hole:** The positive charge vacancy left behind at the atom. It behaves like a virtual positive particle — in reality, neighboring electrons hop sequentially into the vacancy, making the hole appear to move in the opposite direction.

#### E-Field Direction
The electric field points from the **+ plate to the − plate**.
- Electron (−): moves **opposite** to E-field → drifts left ←
- Hole (+): moves in the **same direction** as E-field → drifts right →

Both contributions result in conventional current flowing in the **same direction** (+ → −).

---

### Drift Current

Generated by the ordered motion of carriers under an applied E-field:

```
J_drift = q(n·μₙ + p·μₚ)·E = σ·E
```

| Symbol | Description | Value (300K, intrinsic) |
|--------|-------------|------------------------|
| μₙ | Electron mobility | 1350 cm²/V·s |
| μₚ | Hole mobility | 480 cm²/V·s |
| σ | Conductivity | q·nᵢ·(μₙ + μₚ) |

**Why μₙ > μₚ?**  
Hole movement is actually caused by sequential electron hopping. Because an electron must "wait" for a neighbor to jump, the apparent hole motion is slower.

---

### Mobility and Temperature

Lattice scattering: as temperature increases, Si atoms vibrate more, causing more frequent carrier collisions.

```
μ ∝ T⁻²·³
```

| T | μₙ | μₚ |
|---|----|----|
| 300K | 1350 cm²/V·s | 480 cm²/V·s |
| 450K | ~590 cm²/V·s | ~210 cm²/V·s |

However, the exponential increase in nᵢ outweighs the mobility decrease, so **σ increases overall**.

---

### Doping and Ionized Impurity Scattering

When dopant atoms enter the lattice they become ionized (+/−), creating additional scattering centers (Matthiessen's rule):

```
1/μ = 1/μ_lattice + 1/μ_impurity
```

**Key point:** Ionized impurity scattering significantly affects only the **majority carrier**.

| Mode | Affected | Reason |
|------|----------|--------|
| N-Type | μₙ ↓ | Electrons (majority) frequently encounter P⁺ ions |
| N-Type | μₚ ≈ unchanged | Holes (minority) are so few they rarely interact with ions |
| P-Type | μₚ ↓ | Holes (majority) frequently encounter B⁻ ions |
| P-Type | μₙ ≈ unchanged | Electrons (minority) are so few they rarely interact with ions |

**Mass action law:** n · p = nᵢ²

---

### Diffusion Current

Without an E-field, a concentration gradient drives carriers from high → low density (Fick's Law):

```
J_diffusion = q·Dₙ·(dn/dx) - q·Dₚ·(dp/dx)
```

Diffusion coefficients are derived from mobility via the Einstein relation:

```
Dₙ = μₙ · kT/q     Dₚ = μₚ · kT/q
```

| T | Dₙ | Dₚ |
|---|----|----|
| 300K | ~34.9 cm²/s | ~12.4 cm²/s |
| 450K | ~22.9 cm²/s | ~8.1 cm²/s |

**In diffusion mode:**
- Carriers are concentrated on the left side
- E-field is off; net motion is driven by the gradient
- Generation and recombination are disabled (to observe a steady concentration gradient)
- As particles spread out and equilibrate, net current approaches zero

---

### Total Current Density

```
J_total = J_drift + J_diffusion
        = q(n·μₙ + p·μₚ)·E + q(Dₙ·dn/dx - Dₚ·dp/dx)
```
